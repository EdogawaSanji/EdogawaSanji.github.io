C++ 2.0 新标准

Variadic Templates

initializer_list

  • {} : 构造 initializer_list 的临时对象：底层包含对象的 array 指针
  • C++ 2.0 所有容器的构造、赋值函数、insert、assign 均可以接收 initializer_list；max()、min() 也可接收 initializer_list

构造函数关键字

explicit

  • 主要使用在 构造函数 前；阻止编译器隐式调用构造函数，只能被显式调用

= default

  • 使用编译器默认版本构造函数：不进行任何赋值操作 or 调用父类构造函数

= delete

  • 可用于任何函数，标示函数无法使用

= 0

  • 纯虚函数：父类不定义，交由子类定义

含有指针的 class or struct 需要自行定义 Big-Three

  • Big-Three：构造函数、赋值函数、析构函数

using

Alias Template：模板类别名

  • typedef：不可对模板类生成别名，只可对特定对象 or 类生成别名

  • template template parameter

Type Alias

noexcept

  • ==如果 move constructor or move assignment 非 noexcept，vector 在二倍成长的时候不会调用 move==

override

  • 用于类的虚函数
  • 子类 override 父类的虚函数时，编译器会 check 父类是否有对应的同名虚函数；没有则会报错

final

decltype

  • decltype：= typeof；取得变量 or 对象的类型

lambdas

move

右值

  • 临时对象
  • std::move(对象)

forward

  • 完美转发
